# country-explorer-app
# 🌐 Country Explorer Web Application

## 📌 Project Overview

This project is a responsive web application developed using **HTML, CSS, and JavaScript**. The application integrates a public API to fetch and display real-time country data dynamically. It demonstrates the use of JavaScript concepts, API integration, and interactive user interface development.

Users can explore country information through searching, filtering, and sorting features in a clean and responsive web interface.

---

## 🎯 Project Objective

The objectives of this project are:

- To integrate a public API using JavaScript `fetch()`
- To dynamically render data on a webpage
- To implement Array Higher Order Functions
- To build an interactive and responsive UI
- To practice clean and structured web development

---

## 🔌 Public API Used

**API Name:** REST Countries API  
**API Link:** https://restcountries.com/

The API provides information such as:

- Country Name
- Capital City
- Region
- Population
- National Flag

---

## ✨ Features

- Fetch and display real-time API data
- Search countries by name
- Filter countries based on region
- Sort countries alphabetically
- Responsive design for mobile, tablet, and desktop
- Loading indicator during API requests
- Dark Mode / Light Mode toggle (optional feature)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- Array Higher Order Functions:
  - map()
  - filter()
  - sort()
  - find()
